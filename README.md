# PulseFlow API

A production-style backend service built with FastAPI and Pydantic for ingesting and validating telemetry events.

## Why this project
I built this project to practice backend engineering fundamentals commonly used in real software systems, including:
- REST API design
- Request and response validation
- Modular service structure
- Testing with pytest
- Future database and event-driven integration

## Current features
- Health check endpoint
- Event ingestion endpoint
- Pydantic-based request validation
- Basic pytest test coverage

## Tech stack
- Python
- FastAPI
- Pydantic
- pytest
